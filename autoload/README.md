# 更新日志

插件的更新日志，插件的功能在前面的目录

## actor_to_style

**v1.1.0** 

- 记录【文本导入选项】中的输入并保存，应用到下一次输入

**v1.1.1** 

- 增加判断非UTF-8编码文本添加的功能

**v1.1.2** 

- 修复【文本导入选项】的输入保存

**v1.2.0**

- 添加一个通用配置文件



## clips_printer

**v1.0.0** 

- 读取文本并输出动态遮罩，支持行内标签 (`an` `pos` `fsc` `fscx`/`fscy`)

**v2.0.0** 

- 行内标签旋转(`frz`)和修改旋转中心的旋转(`\org(*,*)\frz`)也添加到支持

**v2.1.0** 

- 边框厚度(`bord`)和阴影距离(`shad`)的改变相应的会改变遮罩的位置

**v2.2.1** 

- 支持绘图代码的动态遮罩：行内标签 (`p`)
- 水平方向添加了左右各5像素的空白，使其在展开时不会过于紧凑

**v2.3.0**

- 添加一个通用配置文件
